# Submission : Proyek Image Generation ✨

## 📑 Deskripsi
Proyek ini adalah submission untuk penilaian praktik Generative AI pada kelas Belajar Penerapan Data Science di Dicoding. Fokus utama proyek ini adalah melakukan sintesis gambar, baik melalui input teks (Text-to-Image) maupun referensi gambar (Image-to-Image), guna memahami bagaimana model AI memproses instruksi visual untuk menghasilkan karya digital yang kreatif dan akurat.

## 🗂 Struktur Proyek
- `Pipeline_Submission_BFGAI_Meakhel-Gunawan.ipynb`: File yang digunakan untuk melakukan eksperimen Generate Image dengan berbagai kriteria yang ada. 
- `Streamlit_submission_BFGAI_Nama-siswa.ipynb`: File Streamlit yang digunakan untuk membuat interface dan melakukan Generate Image dengan parameter yang tersedia.
- `video_demo_aplikasi_BFGAI.mp4`: File video demo singkat dari aplikasi Streamlit dengan format sebagai bukti otentik bahwa aplikasi pernah berjalan lancar.
- `requirements.txt`: File yang berisi requirement yang dibutuhkan dalam menjalankan `Pipeline_Submission_BFGAI_Meakhel-Gunawan.ipynb`
- `README.md`: File dokumentasi.
